# Ragesh — UI/UX Designer Portfolio

A premium, high-performance portfolio website built for **Ragesh**, a professional UI/UX & Product Designer. This project showcases a deep commitment to craft, user-centered design, and technical excellence through modern web technologies.

## ✨ Key Features

- **Interactive Visuals**: Custom-built background particle canvas and a responsive cursor-following ring.

- **3D Interactions**: Immersive 3D tilt effects on project cards and case studies using vanilla JavaScript.

- **Modern Animations**: Smooth scroll-reveal effects and staggered entrance animations for a premium feel.

- **Theme Support**: Seamless Dark and Light mode transitions with custom color tokens.

- **Mobile Optimized**: Fully responsive layout that scales elegantly from high-res monitors to mobile devices.

- **Performance Focused**: Built with pure HTML, CSS, and JS—no heavy frameworks, ensuring near-instant load times.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Properties, Flexbox, Grid)

- **Interactivity**: Vanilla JavaScript (ES6+)

- **APIs used**: 
  - `IntersectionObserver` for scroll animations and navigation active states.
  - `Canvas API` for the dynamic background particle system.
  - `requestAnimationFrame` for high-performance cursor tracking.

## 📂 Project Structure

```bash
├── index.html      # Main markup and content structure
├── style.css       # Design tokens, layouts, and animations
```
